## 自己PR
ユーザーにとって分かりやすいUIを考えることが得意です。
以前、カレンダーを実装した際、日付の選択数が上限に達してもそれ以上選択できないことをユーザーに伝えられていない課題を見つけました。
そこで、上限に達した際にカレンダーをグレーアウトにすることで、ユーザーにそれ以上選択できないことを伝える機能を考え、実際にリリースするに至りました。
今後もユーザーにとって分かりやすい機能を作っていきたいです。
